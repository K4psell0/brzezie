】Dystrybucje systemu Linux【

Dystrybucja Linuksa – uniksopodobny kompletny system operacyjny zbudowany na bazie jądra Linux. Znakomita większość dystrybucji zawiera dużo elementów projektu GNU, co sprawia, że są one dystrybucjami GNU/Linuksa. Dwie głównie linie systemów Linux to Debian i Slackware. Najpopularniejsze dystrybucje Linuxa to:
  • Linux Mint
  • Ubuntu
  • Lubuntu
  • Zorin OS
  • KDE neon
  • Kubuntu
  • Vanilla OS
  • Debian
  • Fedora
  • Manjaro
  • Arch Linux
  • openSUSE
  • CentOS
  • MX Linux
  • elementary OS
  • Gentoo
  • ArcoLinux
  • Solus
  • Deepin
  • Pop!_OS

Cytując portal Linuxiarze.pl: “Powyższe zestawienie najpopularniejszych dystrybucji Linux bazujących na Ubuntu w styczniu 2023 roku ma charakter czysto poglądowy i opiera się na odsłonach stron użytkowników portalu Linuxiarze.pl. W zestawieniu były brane pod uwagę strony dystrybucji, pobierania, poradniki instalacji oraz komentarze”.
