】Najważniejsze katalogi w systemie Linux【

Podobnie jak w Windowsie Linuxy posiadają konkretną strukturę katalogów, która występuje w praktycznie każdej dystrybucji. struktura posiada buudowę drzewiastą, gdzie główny katalog oznaczony jest symbolem "/", od niego wychodzą wszystkie pozostałe katalogi. Do najważniejszych z nich zaliczamy:
  • /bin - pliki wykonywalne oraz skrypty, które mogą być wykonywane przez wszystkich użytkowników systemu (użytkownicy nie potrzebują specjalnych uprawnień)
  • /boot - pliki używane przez program ładujący system oraz pliki zawierające obraz jądra systemu. Głównym boot menagerem w Linuxach jest obecnie GRUB, który wspiera
    bootowanie z różnych nośników i zastąpił starszą werję boot loadera LILO
  • /dev - zawiera pliki specjalne reprezentujące urządzenia, które występują w systemie
