Linux to rodzina uniksopodobnych systemów operacyjnych opartych na jądrze Linux.
System ten został stworzony przez fińskiego programistę Linusa Torvaldsa w 1991 roku. Linux jest jednym z przykładów wolnego i otwartego oprogramowania (FLOSS), co oznacza, że jego kod źródłowy może być dowolnie wykorzystywany, modyfikowany i rozpowszechniany. Linux jest często używany na serwerach, ale także jest dostępny dla użytkowników domowych jako alternatywa dla systemów operacyjnych takich jak Windows i macOS.
