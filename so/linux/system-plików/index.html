】System plików w Linuxie【

Linuxy posługują się własnym systemem plików EXT, który tworzy jedną wspólną całość rozpoczynając się w katalogu głównym. Instalując system możemy zdecydować, że poszczególne jego elementy mogą buć rozmieszczone na osobnych partycjach osobnych fizycznych urządzeń. Przykładowa konfiguracja dzieląca katalogi systemu na różne lokalizacje może wyglądać następująco:

  • SDA2 ➝ /
  • SDB2 ➝ /home
  • SDA1 ➝ /boot







W Linux'ach występuje kilka rodzajów plików:
  • zwykłe pliki oznaczone myślnikiem, do plików tych zaliczamy np. pliki tekstowe ASCII
  • binarne pliki wykonywalne
  • wykonywalne pliki tekstowe
  • pliki z danymi aplikacji
  • specjalne pliki znakowe oznaczone literą C
  • pliki blokowe oznaczone literą D
  • wiązania symboliczne - litera C
